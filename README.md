# Frame Adda Cinematix 🎬

A premium, high-end cinematic portfolio and business website for **Frame Adda Cinematix**. This website is designed to showcase professional filmmaking, wedding cinematography, and commercial production services with a focus on luxury aesthetics and interactive user experience.

![Frame Adda Cinematix Logo](Frame%20Adda%20Cinematix%20-%20Logo.png)

## ✨ Key Features

- **Interactive 3D Background**: A cinematic darkroom lab environment built using **Three.js**, featuring hanging photos and atmospheric lighting.
- **AI Virtual Assistant**: A custom-built AI chatbot to handle client inquiries about wedding films, commercial shoots, and locations.
- **Cinematic Portfolio**: An elegant accordion-style gallery showcasing different categories like Wedding Films, Commercials, and Cinematic Diaries.
- **Professional Contact System**: Integrated with **Web3Forms API** for seamless background email routing (no manual `mailto:` redirects).
- **Mobile Optimized**: Fully responsive "Phone View" with a custom hamburger menu and optimized touch targets.
- **Premium UI/UX**: Glassmorphism cards, custom golden scrollbars, scroll-reveal animations, and a "Camera Flash" logo interaction.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **3D Graphics**: Three.js (WebGL)
- **Icons & Fonts**: FontAwesome 6, Google Fonts (Outfit, Dancing Script, Inter)
- **Email Routing**: Web3Forms API
- **SEO**: Custom OpenGraph metadata and meta description for professional social sharing.

## 🚀 How to Run Locally

1. Clone this repository (or download the files).
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge).
3. Ensure you have an active internet connection for CDN-loaded libraries (Three.js and FontAwesome).

## 🌍 Deployment

This site is a static project and can be easily hosted on:
- **Netlify**: Drag and drop the folder.
- **Vercel**: Connect your GitHub repository.
- **GitHub Pages**: Deploy via the repository settings.
